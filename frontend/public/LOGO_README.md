# Logo

For the best look on the landing page (gradient showing through), use a **PNG with transparent background**:

- Only the logo graphic and "UPHUMAN" letters, **no background**.
- Replace `logo.png` in this folder with your transparent version.
- You can export it from your design tool (e.g. Figma, Photoshop) with "Transparent background" / PNG-24, or use a tool like [remove.bg](https://remove.bg) to remove the background from the current logo.

The app will use `logo.png`; when it has transparency, the page gradient will show through.
